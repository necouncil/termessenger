# terminal messenger

первый проект по криптографии, решил сделать мессенджер с шифрованием.

го знаю плохо поэтому помогал ИИ, но разобрался как работает AES-GCM и почему нельзя переиспользовать nonce.

## запуск

```
go run main.go server
go run main.go client
```

оба вводят одинаковый пароль. если пароли разные — сообщения не расшифруются.

## что внутри

- пароль → ключ через SHA-256
- шифрование AES-256-GCM
- в терминале видно что реально уходит по сети в зашифрованном виде
